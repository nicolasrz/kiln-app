
# KillDash

# How to install
- composer install
- npm i

# How to test
- set "projectId" in resources/js/wagmi-config.ts (given by email)
- npm run build
- cp .env.example .env
- php artisan migrate
- php artisan db:seed (optional)
- php artisan key:generate
- php artisan serve
- go on localhost:8000
